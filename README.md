# Library Management System - MySQL Database

A complete library management system built with MySQL featuring book tracking, member management, loans, and reporting.

## Features
- Complete database schema
- Sample data
- Stored procedures for operations
- Useful views and queries
- Documentation

## Quick Start
1. Run `database/schema.sql` to create database
2. Run `database/sample-data.sql` to populate with sample data
3. Use stored procedures for library operations

## Database Structure
- **Books & Authors Management**
- **Member Management** 
- **Loan & Return System**
- **Reservation System**
- **Fine Management**
